📅 Daily Standup Cycle

⏰ Standup Time: [TBD] IST

Pre-standup report sent 30 minutes before

🌅 Pre-Standup (-30 min)

  • 🎖️ Captain: Query Notion for projects with events in next 7 days
  • ⚒️ Forge: Summarize dev task status per project
  • 🗺️ Atlas: Flag logistics gaps (travel not booked, DC not ready, inventory not blocked)
  • 🔥 Blaze: List pending client follow-ups
  • → Combined report sent to CFT Project Confirmations

🎤 During Standup

  • Each employee shares task updates
  • Which project they're working on
  • Review upcoming events (next 7-14 days)
  • Blockers & dependencies raised
  • Decisions made, action items assigned
  • 🎙️ Fireflies.ai records everything

🌆 Post-Standup (+15 min)

  • 🎖️ Captain: Pull Fireflies transcription
  • Extract action items & decisions
  • Update Notion tasks with assignments
  • Send MoM summary to relevant groups
  • Notify owners of their new tasks
  • Update project status if changed

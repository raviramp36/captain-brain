# 📝 BD Project Confirmation Format

Use this format when posting confirmed projects in **CFT Project Confirmations** group.

---

## Template

```
🎯 PROJECT CONFIRMED

📋 Event: [Event Name]
🏢 Client: [Agency/Direct Client]
📍 Location: [Venue, City]

📦 Products:
- [Product 1] - [Quantity]
- [Product 2] - [Quantity]

📅 Dates:
- Setup: [Date]
- Event: [Date(s)]

💰 Costing: [Confirmed/Pending]
📝 Special Notes: [Any specific requirements]

@PM please add to Notion
```

---

## Example

```
🎯 PROJECT CONFIRMED

📋 Event: Air Cargo Summit
🏢 Client: ARC Worldwide
📍 Location: Jio Centre, Mumbai

📦 Products:
- DNA Display - 1 unit
- AI Photobooth - 2 units

📅 Dates:
- Setup: 24th Feb '26
- Event: 25th - 27th Feb '26

💰 Costing: Confirmed
📝 Special Notes: Client wants custom branding on DNA

@PM please add to Notion
```

---

## Field Guidelines

| Field | Required | Format | Example |
|-------|----------|--------|---------|
| Event | ✅ Yes | Event/Project name | Air Cargo Summit |
| Client | ✅ Yes | Agency or direct client | ARC Worldwide |
| Location | ✅ Yes | Venue, City | Jio Centre, Mumbai |
| Products | ✅ Yes | Product - Quantity | DNA Display - 1 unit |
| Setup Date | ✅ Yes | DD Mon 'YY | 24th Feb '26 |
| Event Date | ✅ Yes | DD Mon 'YY or range | 25th - 27th Feb '26 |
| Costing | ✅ Yes | Confirmed/Pending | Confirmed |
| Special Notes | Optional | Any requirements | Custom branding needed |

## Product Names (Use Exact Names)

**Kinetic:**
DNA Display, Aurora, Nebula, HMRS, Tri-Helix, Matrix Screens, Flying Screens, Kinetic Winch, Origami Wall

**AI Products:**
AI Photobooth, AI Music Generator, AI Bot Greeting, AI Bistro, AI Art, Face Recognition

**Photobooths:**
360 Photobooth, Mirror Booth, Glambot, GIFF Booth, Strip Booth, AR Booth

**Interactive:**
Touch Kiosk, Gesture Screen, Interactive Floor, Interactive Wall, Immersive Room

**Displays:**
LED Cube, Holofan, Anamorphic Display, LED Tunnel, 3D Cube

---

## What Happens After You Post

1. **Captain parses** your message automatically
2. **Notion project created** with all details
3. **Relevant teams notified:**
   - Engineering (if product needs Software/Embedded work)
   - Operations (inventory, travel, logistics)
   - Creative (post-event content planning)
4. **Tracked in daily standup** prep reports

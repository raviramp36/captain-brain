## Workflow Template

For each process, document:
1. **Trigger** — What starts this process?
2. **Steps** — Sequential actions
3. **Owners** — Who does what?
4. **Tools** — Software/systems used
5. **Handoffs** — Where does it go next?
6. **SLAs** — Time expectations
7. **Exceptions** — Edge cases

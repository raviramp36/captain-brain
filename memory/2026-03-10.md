## Key Learning
- Each agent has its OWN `auth-profiles.json` in `/root/.openclaw/agents/[agent]/agent/`
- When API keys expire or change, must update ALL agent auth files, not just the main config
- Universal key sync command pattern:
  ```bash
  for agent in main forge blaze atlas; do
    jq '.profiles["openrouter:default"].key = "NEW_KEY"' \
      /root/.openclaw/agents/$agent/agent/auth-profiles.json > tmp && mv tmp auth-profiles.json
  done
  ```

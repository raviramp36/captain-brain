## 📈 Project Workflow

### Project Stages (Pipeline)
1. **Ideation** - Initial concept
2. **Internal Approval** - Team sign-off
3. **Client Approval** - Client confirmation
4. **Briefing** - Requirements gathering
5. **Assigning** - Team allocation
6. **Demo** - Proof of concept
7. **Execution Planning** - Logistics
8. **Execution Training** - Team prep
9. **Execution** - Event day
10. **Review** - Post-event analysis

### Status Flow
- Yet to confirm → Need to brief → In progress → Setup → Live → Review → Done
- Alternate paths: Paused, Canceled
